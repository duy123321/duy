## Getting Started

### Installation

```bash
bun install
```

### Development

```bash
bun run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
bun run build
```

### Preview Production Build

```bash
bun run preview
```

## Usage

- Use **Arrow Left/Right** keys to navigate between games
- Click on game tiles to select them
- The selected game will be highlighted and scaled up
- Each game tile has a unique color and icon

## Technologies

- React 18
- Vite
- CSS3 (with animations and gradients)
